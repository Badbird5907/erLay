# erLay
erLay is a ultra portable CoreXY 3D printer made by [Evan Yu](https://evanyu.dev) and [Aaron Huang](https://runthebot.me)

## Goals
- Ultra portable (for robotics competitions!!)
- CoreXY
- Pico MMU
- Zero Click ABL
